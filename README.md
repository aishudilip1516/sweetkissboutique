# SweetKissBoutique

##  Deploying the app

-   Run the below command

```
npm run deploy
```

-   Push the changed code

```
git push
```

-   You will notice branch name **gh-pages** in your github repo

-   Go to **Settings**
    -   Navigate to the **Github Pages** section
    -   Select the branch as **gh-pages** and select the folder as **root**
